1. Скачать ollama
https://ollama.com/download
Cкачать OllamaSetup.exe
Запустить установщик
После установки Ollama запустится автоматически
(уже можно работать)

2. Скачать модель
Открыть терминал проверьте:
ollama --version

Скачайть модель (берем маленькую)
ollama pull llama3.2:1b

Запустить чат
ollama run llama3.2:1b
(чат откроется в консоли)

3. Создаем свой чат на React
Забрать репозиторий с GIT:
https://github.com/EugenePokh/ai_internal_llama.git

npm install
npm install axios file-saver react-icons
npm install pdfjs-dist

Запустить
npm run dev

Остановить
Ctrl + C
